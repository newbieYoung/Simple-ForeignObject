# SimpleScreenshot

基于 SVG 的小程序、H5 图片生成组件。

## 特性及优势

- 支持全部 CSS 属性；
- 所见即所得，学习、使用成本低；
- 当客户端支持 SVG 渲染时可以不依赖后台服务。
